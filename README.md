# Leit_Ger_Mens_Excel_Py
Leitura e Geração de Mensagens a partir de Excel

ETL em Python – Leitura e Geração de Mensagens a partir de Excel

Este projeto tem como objetivo demonstrar um processo ETL (Extração, Transformação e Carga) utilizando Python, a partir de um arquivo Excel contendo dados de clientes.

O script realiza a leitura dos dados, gera mensagens personalizadas e salva o resultado em um novo arquivo Excel.
